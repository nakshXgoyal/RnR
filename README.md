# 🌍 RnR – RootsNRoutes

**RnR (RootsNRoutes)** is a tourism web application designed to promote tourism in Jharkhand by showcasing both popular destinations and hidden gems.
It helps users explore places, search destinations, and even hire local guides for a better travel experience.

---

## 🚀 Live Demo

🔗 (https://roots-n-routes.web.app/)
📂 https://github.com/anuragxtiwari/RnR

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend/Services:** Firebase
* **Database:** Firebase Firestore
* **Authentication:** Firebase Auth

---

## ✨ Features

* 🔍 Search destinations by name
* 📍 Explore popular and hidden tourist spots in Jharkhand
* 🧭 Detailed description of each place
* 👨‍🏫 Option to hire local guides
* ⚡ Fast and responsive UI

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/anuragxtiwair/RnR.git
```

### 2. Navigate to project folder

```bash
cd RnR
```

### 3. Install dependencies

```bash
npm install
```

---

## ▶️ Run the project

```bash
npm start
```

App will run on:
👉 http://localhost:3000
